angular-duoshuo
===============

这是一个Angular的插件。    
修改自: https://github.com/kirstein/angular-disqus 。    
Demo: http://static.ricter.info/angular-duoshuo


##用法
在你需要的地方加上`<div duoshuo="id"></div>`，此处id为唯一标识符。    
你需要设置`window.duoshuoQuery`中的`short_name`，例如： 

    {
        short_name: "angular-hentai"
    }
